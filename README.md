# BibliotecaBase

Proyecto base creado desde cero para gestionar un catálogo simple de libros y su flujo de préstamo.

## Uso rápido

```bash
npm test
```

## API disponible

- `Book`: crea una entidad libro con `id`, `title` y `author`.
- `Library`: permite agregar, buscar, listar, prestar y devolver libros.
